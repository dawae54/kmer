Descripción detallada

Implementación del algoritmo de backtracking para resolver el problema de las losetas.

Este programa resuelve el problema de las losetas, donde se busca una secuencia de movimientos que permita pisar cada loseta un número determinado de veces, comenzando desde una posición inicial y terminando en una posición final. El programa utiliza un algoritmo de backtracking para explorar todas las posibles combinaciones de movimientos y encuentra una solución válida si existe.

bool back_recursivo ( Solucion & Sol,
int k )

Función recursiva que implementa el algoritmo de backtracking para encontrar una solución correcta.

Una solución correcta es aquella que termina en la posición de salida tras pisar cada loseta el número de veces que indica.

Parámetros
SolLa posible solución actual.
kEl índice de la acción a procesar.
Devuelve
true Si se ha encontrado una solución.
false Si no se ha encontrado una solución.
